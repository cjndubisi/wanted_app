# Wanted

Wanted react Navtive and Web App

## Requirements

- node@12.15
- Xcode@11
- Cocoapods
- Android Studio

### Installation

```
npm install
```

### Run

```
// ios
$: cd ios
$: pod install
$: cd ..
$: npm run ios

// android
$: npm run android

// web
$: npm run web
```

### Test

Run `npm test`
